# py3d
Python API for visualizing complex network data (and more) using d3.js

## Installation

You can install py3d using pip:

```bash
pip install py3d
```

For development installation:

```bash
git clone https://github.com/yourusername/py3d.git
cd py3d
pip install -e .
```
